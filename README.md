# Oslo Jaipur - Rooftop Lounge & Bar Website


Custom WordPress website developed for Oslo Jaipur, Asia's largest open-air rooftop lounge and bar in Jaipur, Rajasthan (2025 client project).

- **Client Industry**: Hospitality & Nightlife (Rooftop Dining, Bar, Events, Multi-Cuisine Restaurant)
- **My Role**: Full custom WordPress development, theme customization, Elementor-style sections, performance optimization
- **Timeline**: 4-6 weeks
- **Team Size**: Solo developer

## Key Features Implemented
- Dynamic hero section with rotating taglines ("Good Food Good Mood", "Enjoy Food Enjoy Life", "Stay Healthy Stay Strong") and "Book Now" CTAs
- Services showcase: Event Hosting, Diverse Culinary Offerings (North Indian, Chinese, Continental, Oriental, Pizza, Biryani), Open-Air Lounge & Bar, Live Music & Themed Events
- About section highlighting panoramic city views from 7th floor Jaipur Centre Mall, Durgapura; operated by Vervain Hospitality Services LLP
- Stats display: 25+ Handcrafted Drinks, 80%+ Customer Satisfaction, 3K+ Social Engagement, 200+ Event Capacity
- Testimonials carousel with real customer quotes on ambiance, food, and events
- Gallery of rooftop views, nightlife moments, and dining setups
- Contact/Reservation section with "Find Us Here" map link, phone/WhatsApp, and booking form
- Responsive design optimized for nightlife & food visuals (mobile-first)
- SEO-friendly for Jaipur rooftop dining/bar/event searches

## Technical Stack
- WordPress v6.7+
- PHP 8.2+
- Custom theme (likely Grillino or similar from ThemeForest) + Elementor Pro or builder for sections/sliders
- Custom CSS for vibrant styling, hover effects, and rooftop aesthetic
- JavaScript for interactions (testimonials, animations if present)
- Plugins: Contact/reservation forms (WPForms or similar), Yoast SEO, image optimization/cache
- Performance: WebP images, lazy loading for gallery-heavy rooftop photos
